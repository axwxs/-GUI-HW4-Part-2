Multiplication Table Generator

Please enter integer values between -50 and 50 for the maximum and minimum values for both the columns and rows in order to generate a multiplication table

The maximum value for the columns and rows must be greater than the minimum value for the columns and rows. Also any non-integer values will be truncated (ex. 1.5 will be 1).
